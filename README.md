# Ocean.io (ocean-io)

Ocean.io is a B2B account-based prospecting and enrichment platform focused on lookalike search and ICP refinement. It offers a REST API for company enrichment and lookalike account discovery. API access is gated to paying customers and configured per contract.

## APIs

### Ocean.io Enrichment API
REST API for company enrichment given a domain, plus lookalike search returning similar companies based on a seed list. Authentication via API key issued to enterprise customers.
- **Base URL:** `https://api.ocean.io`
- **Docs:** https://ocean.io/help/api
